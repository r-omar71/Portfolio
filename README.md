# Personal Portfolio Website

A responsive personal resume website showcasing my background, projects, and technical skills.

## Overview
This website presents:
- Personal information
- Education
- Selected projects
- Technical skills
- Contact details

The goal is to provide a clear and professional overview of my work and capabilities as a software engineer.

## Technologies
- HTML5
- CSS3
- Bootstrap (selected components)
- Git & GitHub Pages

## Features
- Semantic HTML structure
- External stylesheet architecture
- Responsive layout for mobile, tablet, and desktop
- Accessibility best practices
- Clean and maintainable code
